# Edu Altutors - Learning Management System

A modern LMS built with Next.js, React, and Tailwind CSS.

## Quick Start

```bash
cd edu-altutors-lms
npm install
npm run dev
```

Open http://localhost:3000

## Features

- Dashboard with progress tracking
- Quiz and learning progress indicators
- Calendar widget
- Featured packages
- Testimonials
- Responsive design
